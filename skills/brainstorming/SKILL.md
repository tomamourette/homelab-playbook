# Brainstorming

**Phase:** 1 — Analysis | **Agent:** Analyst (Mary) | **Model:** Gemini Pro (local) | **Cost:** $0

## When to Use

Open-ended ideation and exploration. Generate ideas, explore possibilities, challenge assumptions.

## Execution

Handle locally on Gemini Pro. This is conversational — no rigid structure needed.

1. Understand the topic or problem from the user
2. Generate diverse ideas — quantity over quality at this stage
3. For each idea, briefly note feasibility and effort
4. Group ideas into themes
5. Ask the user which directions interest them
6. Optionally write a summary to `docs/research/brainstorming.md`

## Output

A list of ideas and directions for the user to choose from. This is the most free-form BMAD workflow — follow the user's energy.
